## Introduction

This is a mod developed for Minecraft Java Edition. This mod is committed to make feeding maids more meaningful.

## Features

- The mod keeps track of how many unique foods each maid has eaten. As they pass certain milestone amounts of unique foods, they permanently gain new hearts, additional armor value, additional attack damage value. Also, maids' favorability increases when reaching new milestones.
- Configurable milestones and hearts, armor value, attack damage value, favorability gained per milestone.
- Interact with your maids with a book in hand to view what types of food have they eaten.
